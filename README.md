# Frontend of Home Exchange Manager Website

## Run
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

## Tech stack
- Vue.js
- Axios
- Element-UI
- Webpack
- Babel
